University of Bristol

Actuator Characterisation

What?

  • I modelled a linear actuator using Finite Element Method Magnetics (FEMM), Simulink and analytical models
  • Each model was used to characterise the actuator and compared in a formal report

How?

  • I used MATLAB scripts to build and automate data collection from the FEMM and Simulink models
  • The analytical model was developed from fundamental magnetic equations in lectures
  • All solving and data analysis was done in Python using Numpy, Scipy, and Matplotlib

Results

  • I built the Simulink model from my understanding of the problem as an extension from the lab script given as guidance; I concluded this provided the best compromise between run time and accuracy

Switched Mode Power Supply

What?

  • My lab partner and I built and tested a 36V AC to a 12V DC SMPS
  • I designed parts of the SMPS including the number of turns on transformers, the size of heatsinks on the MOFETS, and the reactances required for smoothing filters
  • The design was meant for powering 12V water pump from a PV array

How?

  • Design calculations and optimisations were done by hand calculations and using Simulink to model the SMPS
  • Testing was done at each stage of PCB assembly to ensure everything worked

Results

  • We were one of only a handful of pairs who were able to get the SMPS finished and working in the allotted time frame
