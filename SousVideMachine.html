Sous Vide Machine

What?

  • The goal of this project was to make a temperature-controlled water bath to cook steak in
  • There have been many versions of this project throughout the years with various improvements

How?

  • The first version was built in an old cool box with two mains-powered heaters, an Arduino, a temperature sensor, a relay and an LCD
  • The second version was made using the same electronics in a smaller laser-cut box with a motor for mixing the water
  • The third and current version has been changed to a 12V heated with an added fuse and pre-built thermal switch

Results

  • The first version took 40+ minutes to heat up the water as the cool box was too large. The water temperature was also uneven due to the heaters being in the lid
  • Version one successfully cooked an egg
  • The second version contained PLA parts which have a glass transition temperature on the border of sous vide temperatures
  • My understanding of electrical safety has improved since making version two. I decided to change the heater from mains to 12V due to the lack of proper electrical enclosures
  • This led to version three, which I have much more confidence in as my engineering skills have improved greatly since version one
  • Version three has been assembled and turned on but still needs to be thoroughly tested
